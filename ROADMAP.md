# Roadmap

This document outlines the future plans and vision for the Portfolio Templates Collection.

## Short-term Goals (Q1-Q2 2026)

### New Templates
- [ ] Add React-based portfolio template
- [ ] Add Vue.js portfolio template
- [ ] Add minimal/minimalist portfolio template
- [ ] Add photography portfolio template
- [ ] Add designer portfolio template

### Improvements
- [ ] Add live preview screenshots for all templates
- [ ] Create interactive demo site
- [ ] Improve documentation with video tutorials
- [ ] Add template customization guides
- [ ] Implement automated testing

### Features
- [ ] Add dark mode toggle to all templates
- [ ] Improve accessibility compliance (WCAG 2.1 AA)
- [ ] Add internationalization support
- [ ] Optimize performance and loading times
- [ ] Add SEO optimization guides

## Mid-term Goals (Q3-Q4 2026)

### Templates
- [ ] Add Next.js portfolio template
- [ ] Add Gatsby portfolio template
- [ ] Add portfolio builder tool
- [ ] Add template marketplace

### Infrastructure
- [ ] Set up CI/CD pipeline
- [ ] Add automated deployment examples
- [ ] Create template testing framework
- [ ] Build component library

### Community
- [ ] Create contributor badges
- [ ] Host monthly community calls
- [ ] Establish template design guidelines
- [ ] Create showcase page for user portfolios

## Long-term Vision (2027+)

- Build a comprehensive portfolio platform
- Create portfolio generator with customization options
- Develop premium templates
- Establish partnerships with hosting providers
- Create educational content and courses

## How to Contribute

Have ideas for the roadmap? We'd love to hear them! Please:
1. Open an issue with the `enhancement` label
2. Join discussions in existing roadmap issues
3. Submit PRs for roadmap items you'd like to work on
